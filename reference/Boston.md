# Boston Dataset

The Boston Housing Dataset, derived from information from the U.S.
Census Service.

## Usage

``` r
Boston
```

## Format

### `Boston`

A data.frame with 506 rows and 14 columns:

- x1-x13:

  The covariates in the data

- y:

  The response variable

## Source

Harrison, D. and Rubinfeld, D.L. \`Hedonic prices and the demand for
clean air', J. Environ. Economics & Management, vol. 5, 81-102, 1978
