# Weather Dataset

A subset of data collected as part of the GES DISC datasets for
calibrated brightness temperatures.

## Usage

``` r
Weather
```

## Format

### `Weather`

A data.frame with 2000 rows and 3 columns:

- theta:

  Polar angle

- phi:

  Azimuthal angle

- Y:

  Response variable
