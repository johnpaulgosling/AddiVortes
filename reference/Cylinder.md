# Cylindrical Dataset

Synthetic data generated for cylindrical parameters. The response is
given by the function f(z, theta) = sin(z)cos(theta) + z\*sin(theta)^2,
with added noise.

## Usage

``` r
Cylinder
```

## Format

### `Cylinder`

A data.frame with 400 rows and 3 columns:

- z:

  Height

- theta:

  Polar angle

- Y:

  Response
